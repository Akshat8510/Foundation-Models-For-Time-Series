# Foundation-Models-For-Time-Series📈

A comprehensive repository dedicated to mastering the "GPT Moment" of Time Series. This project tracks my journey from understanding the theoretical architecture of TSFMs to implementing state-of-the-art **Inference Scaling** techniques.

## 🚀 Project Goals
1. **Research:** Analyze the 7 most influential TSFM papers.
2. **Implement:** Build a unified codebase in Google Colab for Zero-shot forecasting.
3. **Innovate:** Explore Inference-time compute scaling (Diversified Inference).
